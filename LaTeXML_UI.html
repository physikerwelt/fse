How to install LaTeXML

Submitted by schubotz on Tue, 05/21/2013 - 23:37
This is a short guide how to install LaTeXML on debian as a web service so that you get the final result of http://latexml.formulasearchengine.com. For a version without user interface use this guide: /LaTeXML There is a manual available at https://svn.mathweb.org/repos/LaTeXML/branches/arXMLiv/webapp/INSTALL here a part of my history sudo apt-get install texlive subversion libclone-perl libdata-compare-perl libio-prompt-perl perlmagick libparse-recdescent-perl libxml-libxml-perl libxml-libxslt-perl libarchive-zip-perl unzip apache2 libapache2-mod-perl2 libplack-perl curl libio-string-perl libfile-which-perl libjson-xs-perl make svn co -r 3364 https://svn.mathweb.org/repos/LaTeXML/branches/arXMLiv LaTeXML/ cd LaTeXML/ sh -c "curl -L cpanmin.us | perl - Mojolicious" perl Makefile.PL make make install chgrp -R www-data /home/LaTeXML/webapp chmod -R g+w /home/LaTeXML/webapp cd /etc/apache2/sites-available/ vi latexml a2ensite latexml apache2ctl restart and my latexml config file: ServerName latexml.formulasearchengine.com DocumentRoot /home/LaTeXML/webapp PerlSwitches -I/home/LaTeXML/lib/ SetEnv PERL5LIB /home/LaTeXML/lib/ $ENV{PLACK_ENV} = 'production'; $ENV{MOJO_HOME} = '/home/LaTeXML/webapp'; SetHandler perl-script PerlHandler Plack::Handler::Apache2 PerlSetVar psgi_app /home/LaTeXML/webapp/ltxmojo ErrorLog /var/log/apache2/latexml.error.log LogLevel warn CustomLog /var/log/apache2/latexml.access.log combined
